# News Page

A simple responsive news page with tab navigation for Life, Sports, Weather, and Nature news. Supports dark and light mode. Optimized for both desktop and mobile devices.

## Features
- Tabbed navigation for different news categories
- Responsive design (mobile-friendly)
- Light/Dark mode toggle
- Clean and modern UI

## Usage
Open `index.html` in your browser.

## Structure
- `index.html` – Main HTML file
- `app.css` – Styles
- `app.js` – JavaScript for tab switching and theme toggle
- `sport-news1.html` – Example news page
